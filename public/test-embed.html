以下展示如何將翻譯工具嵌入到您的網站中

📱 嵌入效果預覽

這是嵌入到客戶網站時的樣子（embed 模式 + 深色主題）

📡 postMessage API 接收翻譯結果

iframe 翻譯完成時會透過 postMessage 傳送結果到父頁面：

等待翻譯結果...

📋 嵌入代碼

將以下代碼貼到您的網站 HTML 中即可使用：

<!-- AI 語音翻譯外掛 --> <div style="width:100%;max-width:420px;height:700px;margin:0 auto;border-radius:16px;overflow:hidden;box-shadow:0 4px 20px rgba(0,0,0,0.15);"> <iframe src="YOUR_DEPLOY_URL/?mode=embed" style="width:100%;height:100%;border:none;" allow="microphone; clipboard-write" title="AI 語音翻譯" ></iframe> </div> <!-- 接收翻譯結果（可選） --> <script> window.addEventListener('message', function(e) { if (e.data && e.data.type === 'TRANSLATION_RESULT') { console.log('翻譯結果:', e.data.payload); // e.data.payload = { source, translation, detectedLang, targetLang } } }); </script>

🎮 控制 API（可選）

從父頁面控制 iframe 的目標語言：

// 設定目標語言 const iframe = document.getElementById('translator-iframe'); iframe.contentWindow.postMessage({ type: 'SET_TARGET_LANG', payload: { lang: 'ja' } }, '*');
